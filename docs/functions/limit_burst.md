# function: limit_burst

Rate limit packets with burst

## Signature

```text
limit_burst(rate number, unit string, burst number) string
```

## Arguments

1. `rate` (Number) Rate value
2. `unit` (String) Time unit
3. `burst` (Number) Burst value
